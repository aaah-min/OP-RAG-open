# Aggregated manuscript results

These files preserve only aggregate, non-identifying results reported for the restricted 50-case manuscript evaluation. They are included for transparency and comparison, not as a substitute for patient-level data.

- `paper_all_modes_summary.json`: aggregate G0–G4 results from the final unified 50-case run.
- `paper_g4_summary.json`: aggregate full G4 run summary.

No case-level input, report, context, or result is present in this directory. Consequently, users cannot independently recompute manuscript-level statistics from this public package.
